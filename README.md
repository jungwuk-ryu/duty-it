## 프로젝트 의의
2025년 05월 기준 국내에는 아직 간호 대학생과 간호 업계 종사자를 대상으로 하는 행사 큐레이션 앱이나 서비스가 없거나 매우 열악합니다. 듀잇은 이러한 문제점을 발견하고 시작된 프로젝트입니다.

## 듀잇은

듀잇은 간호학과와 관련된 행사(학술대회, 공모전, 웨비나 등)를 하나의 앱에서 검색하고 필터링 할 수 있도록 기능을 제공합니다. 사용자는 관심 있는 행사를 앱 내의 캘린더에 추가하고 관리할 수 있습니다. 이 캘린더 기능에 추후 근무표 입력 기능을 도입하여 근무표 관리 앱의 한 부분이 되는 것도 목표로 하고 있습니다.

궁극적으로 국내 간호학의 발전과 간호 서비스 질 향상 그리고 간호사 대우 개선에 기여하는 것을 목표로 합니다.
